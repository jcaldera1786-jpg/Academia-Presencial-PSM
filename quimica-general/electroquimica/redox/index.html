Ejercicios: Redox y Electroquímica

1 Indica cuales de las siguientes reacciones (sin ajustar) son de tipo rédox:

  1. a) \(\text{CaO} + \text{H}_2\text{O} \longrightarrow \text{Ca(OH)}_2\)
  2. b) \(\text{Ag} + \text{HNO}_3 \longrightarrow \text{AgNO}_3 + \text{NO}_2 + \text{H}_2\text{O}\)
  3. c) \(\text{Cl}_2 + \text{H}_2\text{O} \longrightarrow \text{HCl} + \text{HClO}\)

Indica, en aquellas que lo sean, cuál es el agente oxidante y cuál es el reductor.

2 Di el estado de oxidación de cada átomo de:

a) \(\text{NO}_2^-\); b) \(\text{HIO}_3\); c) \(\text{TeF}_6^{2-}\); d) \(\text{N}_2\text{O}_3\); e) \(\text{Na}_2\text{MoO}_4\); f) \(\text{RuF}_5\); g) \(\text{HCO}_3^-\); h) \(\text{S}_2\text{O}_3^{2-}\); i) \(\text{ClO}_4^-\); j) \(\text{CaC}_2\text{O}_4\).

3 Clasifica cada una de las siguientes semirreacciones como oxidación o reducción:

  1. a) \(\text{Ca}(s) \longrightarrow \text{Ca}^{2+}(aq)\)
  2. b) \(\text{Fe}^{3+}(aq) \longrightarrow \text{Fe}^{2+}(aq)\)
  3. c) \(\text{Cl}_2(g) \longrightarrow \text{ClO}_3^-(aq)\)
  4. d) \(\text{OH}^-(aq) \longrightarrow \text{O}_2(g)\)
  5. e) \(\text{NO}_3^-(aq) \longrightarrow \text{NO}(g)\)

4 El cloro y sus compuestos presentan estados de oxidación −1, +1, +3, +5 y +7. ¿Cuáles de las siguientes especies pueden actuar como agentes oxidantes? ¿Cuáles como agentes reductores?

a) \(\text{HClO}_2\); b) \(\text{ClO}_4^-\); c) \(\text{Cl}^-\); d) \(\text{ClO}^-\); e) \(\text{Cl}_2\text{O}_7\).

5 Para cada una de las siguientes reacciones, identifica la especie oxidada, la especie reducida, el agente oxidante y el agente reductor. Ajusta las ecuaciones.

  1. a) \(\text{Fe}^{3+}(aq) + \text{I}^-(aq) \longrightarrow \text{Fe}^{2+}(aq) + \text{I}_2(s)\)
  2. b) \(\text{ClO}_3^-(aq) + \text{S}^{2-}(aq) + \text{H}_2\text{O}(l) \longrightarrow \text{Cl}^-(aq) + \text{S}(s) + \text{OH}^-(aq)\)
  3. c) \(\text{Cr}_2\text{O}_3(s) + \text{Al}(s) \longrightarrow \text{Cr}(s) + \text{Al}_2\text{O}_3(s)\)

6 Ajusta las siguientes ecuaciones en disolución básica:

  1. a) \(\text{MnO}_4^-(aq) + \text{Cl}^-(aq) \longrightarrow \text{MnO}_2(aq) + \text{ClO}^-(aq)\)
  2. b) \(\text{NO}_2^-(aq) + \text{Br}_2(l) \longrightarrow \text{NO}_3^-(aq) + \text{Br}^-(aq)\)

7 Ajusta las siguientes ecuaciones:

  1. a) \(\text{ClO}^-(aq) + \text{CrO}_2^-(aq) \longrightarrow \text{CrO}_4^{2-}(aq) + \text{Cl}^-(aq)\) (medio básico).
  2. b) \(\text{KClO}_3(aq) + \text{H}_2\text{C}_2\text{O}_4(aq) \longrightarrow \text{ClO}_2(g) + \text{CO}_2(g) + \text{H}_2\text{O}(l) + \text{K}_2\text{C}_2\text{O}_4(aq)\)
  3. c) \(\text{Ag}_2\text{S}_2\text{O}_3(aq) + \text{H}_2\text{O}(l) \longrightarrow \text{Ag}_2\text{S}(s) + \text{SO}_4^{2-}(aq) + \text{Br}^-(aq)\) (medio ácido).
  4. d) \(\text{Bi}(s) + \text{HNO}_3(aq) \longrightarrow \text{Bi}_2\text{O}_5(s) + \text{NO}(g) + \text{H}_2\text{O}(l)\)
  5. e) \(\text{Mn}^{2+}(aq) + \text{S}_2\text{O}_8^{2-}(aq) + \text{H}_2\text{O}(l) \longrightarrow \text{MnO}_4^-(aq) + \text{SO}_4^{2-}(aq)\) (medio ácido).
  6. f) \(\text{H}_2\text{O}_2(aq) + \text{Fe}^{2+}(aq) \longrightarrow \text{H}_2\text{O}(l) + \text{Fe}^{3+}(aq)\) (medio ácido).
  7. g) \(\text{FeSO}_4(aq) + \text{KMnO}_4(aq) + \text{H}_2\text{SO}_4(aq) \longrightarrow \text{Fe}_2\text{(SO}_4)_3(aq) + \text{K}_2\text{SO}_4(aq) + \text{MnSO}_4(aq) + \text{H}_2\text{O}(l)\)
